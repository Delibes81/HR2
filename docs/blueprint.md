# **App Name**: Fuerza Vital

## Core Features:

- Catálogo: Catálogo de Productos: Muestra una lista de productos de fitness disponibles para la compra.
- Página de Producto: Página de Producto: Detalla la información de cada producto, incluyendo descripción, precio, imágenes, y opciones de compra.
- Carrito: Carrito de Compra: Permite a los usuarios agregar productos seleccionados y gestionar su compra.
- Checkout: Checkout Simplificado: Un proceso de compra ágil y seguro para completar la orden.
- Recomendaciones AI: Recomendación de Productos: Sugiere productos basados en las tendencias del catálogo usando a generative AI tool, impulsando el descubrimiento y aumentando las ventas.
- Presentación: Página de Presentación: Proporciona información sobre la empresa, su misión, y sus valores.
- Admin: Panel de Administración: Permite gestionar el catálogo de productos, editar la información y actualizar los precios.

## Style Guidelines:

- Color primario: Azul deportivo (#29ABE2) para transmitir energía y confianza.
- Color de fondo: Gris claro (#F0F2F5) para un aspecto moderno y limpio.
- Color de acento: Naranja vibrante (#FF8C00) para llamadas a la acción y destacar elementos importantes.
- Fuente para títulos: 'Space Grotesk' (sans-serif) para un estilo moderno y tecnológico.
- Fuente para cuerpo del texto: 'Inter' (sans-serif) para asegurar legibilidad y claridad.
- Iconos modernos y minimalistas relacionados con el fitness y el deporte.
- Diseño limpio y estructurado con espacios amplios para facilitar la navegación.
